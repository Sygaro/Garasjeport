📘 Hjelp

Dette systemet lar deg styre og overvåke én eller flere garasjeporter via webgrensesnitt.

🧭 Navigasjon

  • Forside: Status og styring av porter
  • Admin: GPIO-konfig, kalibrering, innstillinger
  • Backup: Se, last ned eller gjenopprett backup
  • Logg: Se hendelser, filtrer etter type, port og dato

⚙️ Portstyring

Trykk på Åpne-knappen for å aktivere et relé som styrer porten. Statusindikatorene leser fra sensorer og viser om porten er åpen, lukket, eller i bevegelse.

🔧 Kalibrering

Systemet kan måle tiden det tar for porten å åpne/lukke og lagre dette i konfigurasjonen. Bruk manuelt eller automatisk måling under "Admin → Kalibrering".

📦 Backup

Systemet lagrer konfigurasjonen ved backup. Du kan laste ned eller gjenopprette enkeltfiler ved behov. Automatisk sletting skjer etter X dager.

🔒 Innlogging

Du kan aktivere/deaktivere innlogging i config.json:

{
  "auth": {
    "enabled": true,
    "username": "admin",
    "password": "hemmelig"
  }
}

📢 Feil eller spørsmål?

Se systemlogg via Logg-menyen, eller kontakt utvikler.
