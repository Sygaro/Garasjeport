❓ Hjelp & dokumentasjon

Dette systemet lar deg kontrollere én eller flere garasjeporter via Raspberry Pi. Du kan åpne, lukke, se status og konfigurere porter via et webgrensesnitt.

🔐 Innlogging

Du må være logget inn for å bruke systemet dersom dette er aktivert i config.json. Innlogging beskytter tilgang til porter og innstillinger.

🚪 Portkontroll

  • Status: Viser om porten er åpen, lukket eller ukjent
  • Manuell styring: Åpne eller lukk porter via webgrensesnittet
  • Sensorer: To sensorer per port angir status

⚙️ Innstillinger

  • GPIO-konfig: Velg hvilke pinner som brukes til relé og sensorer
  • Kalibrering: Automatisk måling eller manuell inntasting av åpne-/lukketid
  • Logger: Viser hendelser og feil i systemet
  • Backup: Du kan lagre, gjenopprette og sammenligne config.json

📁 Filstruktur

  • config.json – lagrer porter, kalibrering og innstillinger
  • logs/events.log – strukturert loggfil for hendelser
  • backups/ – katalog for konfigurasjonskopier

🔧 Tips

  • Bruk sudo reboot etter store endringer i maskinvareoppsett
  • Du kan integrere med Homey eller andre systemer via REST API
  • Sett opp releer med pulser på 0.5 sek eller kortere for driftssikker åpning/lukking

🧠 GPIO-pinner på Raspberry Pi

Dersom du kobler releer og sensorer til Raspberry Pi, er det viktig å bruke riktige GPIO-pinner.

Du finner interaktiv oversikt med forklaringer på pinout.xyz.
